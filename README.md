maven-sample
====

## Description

Mavenを利用したJavaアプリケーション作成サンプルです。

### ディレクトリ構造
```
.
├── batch (バッチ)
├── business (共通ロジック)
├── web (Web共通ロジック)
├── web-admin (管理画面)
├── web-front (フロントAPP)
├── assembly.xml
└── pom.xml
```
